About

This app showcases how ServiceStack's API First Development model is able to support HTML Rendered and Ajax Client forms, multiple View Engines, multiple layouts - all utilizing the same Service and declarative Fluent Validation with support for multiple form rendering strategies.

From pure Server HTML generated server controls, flexible decorated HTML controls to client Ajax forms with built-in auto-binding available in either built-in jQuery plugin or dependency-free "Vanilla JS" TypeScript library that's easily consumed from either TypeScript or JavaScript Apps that seamlessly integrates with high-level UI frameworks like Vuetify.

All productive development options also have access to the rich ecosystem around ServiceStack Services including Add ServiceStack Reference for developing end-to-end Typed APIs for all popular Mobile and Desktop platform. This app makes use of the TypeScript support to generate its Typed dtos.ts - providing TypeScript code-bases with compile-time safety and JavaScript code-bases from rich intell-sense from IDEs that can use TypeScript declarations.
